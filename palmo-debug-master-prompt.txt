## FAZA 4: ROOT CAUSE ANALYSIS (15 min)

Jeśli testy failują, systematycznie sprawdzaj:

### A. Backend Routes Registration
```typescript
// Check src/server.ts
- [ ] app.use(cors()) — PRZED routes!
- [ ] app.use(express.json()) — presence
- [ ] app.use('/api/submissions', submissionsRouter) — registered
- [ ] Error handler at END — present?
```

### B. Controllers Logic
```typescript
// Check src/controllers/submissionsController.ts
- [ ] getAll() — exists? returns Array?
- [ ] create() — validates input?
- [ ] Error handling — try/catch?
- [ ] Return format — { success, data }?
```

### C. Database Mock vs Real
```typescript
// Check if using Map or MySQL
const submissionsMap: Map<number, DeliverySubmission> = new Map();
// IF Map-based: DATA LOST ON RESTART!
// FIX: Add persistence or use real MySQL
```

### D. CORS Issues
```typescript
// Should be:
app.use(cors({
  origin: process.env.FRONTEND_URL || 'http://localhost:5173',
  credentials: true,
}));

// Check headers in response:

```

## FAZA 5: FIXES & PATCHES (25 min)

### Fix 1: If Route Not Registered
```typescript
// server.ts
const submissionsRouter = require('./routes/submissions');
app.use('/api/submissions', submissionsRouter);
console.log('✅ Routes registered');
```

### Fix 2: If CORS Missing
```typescript
// Add before routes
const cors = require('cors');
app.use(cors({
  origin: 'http://localhost:5173',
  credentials: true,
}));
```

### Fix 3: If Mock DB Losing Data
```typescript
// Add persistence
const fs = require('fs');
function saveMockDB() {
  fs.writeFileSync('db.json', JSON.stringify(Array.from(submissionsMap.entries())));
}
function loadMockDB() {
  if (fs.existsSync('db.json')) {
    submissionsMap = new Map(JSON.parse(fs.readFileSync('db.json')));
  }
}
```

### Fix 4: If Email Not Sending
```typescript
// emailService.ts — test SMTP
const transporter = nodemailer.createTransport({
  host: 'localhost',
  port: 1025, // MailHog SMTP
  logger: true,
  debug: true
});
```

### Fix 5: If WordPress Sync Failing
```php
// sync.php — add logging
error_log('SYNC START: ' . json_encode($_POST));
error_log('Backend URL: ' . get_option('palmo_backend_url'));
error_log('Response: ' . $body);
```
